# ADAS Project Scope

## Goal
Build an AI-powered Advanced Driver Assistance System that analyzes driving footage and provides safety-related insights.

## Core Features
- Video processing
- Lane detection
- Vehicle detection
- Pedestrian detection
- Lane departure warnings
- Collision risk warnings

## Stretch Features
- Driver safety score
- Dashboard analytics
- Weather detection
- Voice alerts

## Not Building
- Self-driving car
- Vehicle control
- Custom neural networks
- Hardware integration (for now)
